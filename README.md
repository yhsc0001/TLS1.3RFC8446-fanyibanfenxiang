# TLS 1.3 (RFC8446) - 翻译版

## 概览

本仓库提供的是TLS 1.3协议（RFC8446）的中文翻译版文档。TLS（Transport Layer Security）协议是互联网上广泛采用的安全协议，用于在两个通信应用程序之间提供保密性和数据完整性。TLS 1.3是最新的版本，相较于先前的版本，它在安全性能和握手效率上有了显著提升，旨在减少延迟，增强加密算法的标准性，并加强安全性。
